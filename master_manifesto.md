# Quadbit Master Manifesto

This manifesto consolidates the canonical off-server ingestion seed list and guiding context for the Quadbit learning phase. Use it as the single source of truth when seeding or auditing ingestion.

## Purpose
- Provide a deduplicated, categorized list of high-value sources.
- Cover legal, civic, scientific, educational, cultural, and large open corpora domains.
- Serve as the master handoff artifact for ingestion and monitoring.

## Usage Notes
- All entries are HTTPS/HTTP GETtable roots unless noted otherwise.
- Favor robots.txt compliance and rate limits appropriate to each host.
- Track reachability and latency separately; this document is source catalog only.
